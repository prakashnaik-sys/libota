LibOTA is a free ebook reader application.

Features

  • EPUB & PDF reader

Content Sources

  • Project Gutenberg
  • Feedbooks Public Domain
  • Standard Ebooks

LibOTA does NOT host copyrighted content.
All books are provided via public APIs.
Some books may require borrowing via Internet Archive.
